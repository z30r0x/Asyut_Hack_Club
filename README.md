# Asyut Hack Club Website

A simple two-page static website for **Asyut Hack Club** — a local chapter of the global [Hack Club](https://hackclub.com) nonprofit network, built by high school students for high school students.

## Pages

|------|-------------|
| `index.html` | Landing page with a hero section and club introduction |
| `about_us.html` | Team page showcasing all 12 members with hover overlays |

## Styles

| File | Description |
|------|-------------|
| `home.css` | Styles for the home page |
| `about_us.css` | Styles for the About Us page |

---

## Features

- **Responsive layout** using Flexbox and `flex-wrap`
- **Hero section** with a full-cover background image and dark overlay
- **Team cards** with hover-reveal overlays showing name, role, and social links
- **Shared header & footer** across both pages with navigation and social icons
- Bootstrap 5 and Font Awesome 6 used on the About Us page

---

## Getting Started

No build tools or dependencies required. Just open `https://z30r0x.github.io/Asyut_Hack_Club/` in any browser.
